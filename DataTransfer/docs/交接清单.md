## 5. 接手注意事项(易踩坑)

1. **错误码不要再私自挪用编号**:`internal/errors/codes.go` 与接口文档 5.2 一一对应,新增场景用 006 起的扩展号并登记 CSC-005。
2. **配置 JSON 契约已锁定 snake_case**:改 `internal/config/config.go` 字段名/标签 = 破坏 CSC-002 契约,必须走跨服务变更。
3. **遗留预留代码勿删**:`internal/storage`(StorageBackend 信创预留)、`internal/security` TLSProvider 预留、`internal/connector/sidecar`(插件隔离 mock)、opcua native `Subscribe` 空实现、`internal/northbound/mqtt/v5_spike_test.go`。
4. **Devices() 返回深拷贝**是契约(防并发读写共享 protobuf),新 Connector 实现必须遵守。
5. 反射等调试面一律挂 `environment` 开关,默认 production 拒绝。
